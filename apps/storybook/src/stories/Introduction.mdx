## Introduction

This is the Openlane component library — a visual, interactive reference for all shared UI components used across the console application. Each story documents a component's variants, props, and behavior in isolation.

Browse components using the sidebar. All components support light and dark mode via the theme toggle in the toolbar.

## Layout

The sidebar is organized into the following sections:

**Styling & Theming** — Color palette and typography reference.

**Actions** — Interactive controls: Button, CancelButton.

**Data** — Table and data visualization components: DataTable, Table, LineChart, ProgressCircle, and shared table cell components (BooleanCell, DateCell, TagsCell, UserCell).

**Display** — Presentational elements: Avatar, Badge, CustomTypeEnumChip, Info, Kbd, Logo, ObjectsChip, Paragraph, Tag.

**Feedback** — Status and notification components: Alert, Callout, Loading, MessageBox.

**Forms** — Form controls: Calendar, CalendarPopover, Checkbox, Input, InputOTP, Label, MultipleSelector, PasswordInput, RadioGroup, Select, Slider, Switch, Textarea.

**Layout** — Structural containers: Card, Panel, Separator.

**Navigation** — Navigation patterns: Breadcrumb, PageHeading, StepIndicator, Tabs.

**Overlays** — Floating UI: AlertDialog, ConfirmationDialog, Dialog, DropdownMenu, Popover, Sheet, Tooltip.
